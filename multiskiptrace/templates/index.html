How It Works
With Owner Name:

Searches by name + location. Returns ALL matching profiles without address filtering.

Without Owner Name:

Searches by address only. Only returns profiles where current address matches searched address.

Address Expansion:

Addresses like "100-200 Main St" are expanded into individual addresses (100, 101, 102... 200).

Restrictions
  • Address, City, State are required fields
  • Empty Owner Name = address-only search with strict matching
  • Profiles with mismatched addresses are skipped (address-only mode)
  • Already scraped URLs are deduplicated globally
  • Maximum file size: 16MB
